# 🧠 BODMAS Master – Interactive Math Game

## 🚀 Project Overview

**BODMAS Master** is a web-based interactive math game designed to help users practice and understand the **BODMAS (Order of Operations)** rule in a fun and engaging way.

The game dynamically generates arithmetic expressions and challenges users with multiple-choice questions under a time limit, improving both speed and accuracy.

---

## 🎯 Features

### 🔢 Core Features

* Dynamic question generation based on BODMAS rules
* Multiple Choice Questions (MCQs) with 4 options
* Instant answer validation (Correct / Wrong feedback)
* Continuous gameplay without page reload

### ⚡ Advanced Features

* 🎯 Score system (+10 for correct answers)
* ⏱ Timer (10 seconds per question)
* 🎚 Difficulty Levels:

  * Easy → Addition & Subtraction
  * Medium → Includes Multiplication
  * Hard → Full BODMAS with Brackets
* 📊 Progress bar & question tracker
* 🎯 Final result screen with score summary
